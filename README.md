# flipcountdown_zepto
### Description
    Plug in based on jQuery.flipcoutdown.js, usage and jQuery.flipcoutdown.js agreement.

### Example
```js
$(function() {
    var dt = new Date();
    $('#retroclockbox6').flipcountdown({
        beforeDateTime: '8/01/2020' + ' 00:00:01',
        size: 'sm'
    });
})
```
